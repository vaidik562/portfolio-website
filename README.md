PART A - ASSESSMENT TASK

Description:
This is a personal portfolio website created using HTML5, CSS3, and JavaScript. 
The website showcases professional profile, work experience, skills, projects, education, 
and contact information.

Project Name:
Vaidik Ramani - React Native Developer Portfolio

Files Included:
1. index.html - Main HTML file containing the structure of the website
2. styles.css - CSS file for styling and responsive design
3. script.js - JavaScript file for interactive features and functionality
4. README.md - This file

Website Sections:
1. Navigation - Header with navigation menu linking to all sections
2. Home - Landing section with introduction
3. About - Profile information and professional summary
4. Skills - Technical skills and expertise
5. Experience - Work experience and projects
6. Projects - Detailed project descriptions
7. Education - Educational background and qualifications
8. Contact - Contact information and ways to reach out

Technologies Used:
- HTML5 - Semantic markup and structure
- CSS3 - Styling, layout, and responsive design
- JavaScript - Interactive features and functionality
- Google Fonts - Typography

Features:
- Responsive design for mobile, tablet, and desktop devices
- Smooth navigation with scrolling
- Interactive elements and animations
- Professional layout and modern design
- Easy to update content and information

How to Use:
1. Extract all files to a folder
2. Open index.html in a web browser
3. Navigate through different sections using the menu
4. Customize content by editing the HTML and CSS files as needed

Browser Compatibility:
- Chrome
- Firefox
- Safari
- Edge
- Mobile browsers
